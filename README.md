# ZigZag-Conversion
Given a String and number of Rows, convert the string to a zig zag pattern


input: "paypalishiring" , 3
output: "pahnaplsiigyir"

=>

row 1: p   a   h   n                                                                                                                     
row 2: a p l s i i g                                                                                                   
row 3: y   i   r
